GRANT REVOKE COMMIT ROLLBACK

GRANT

The GRANT command is used to grant privileges to users:

  • GRANT SELECT ON table_name TO user_name; - Grants the SELECT privilege on a table to a user.
  • GRANT INSERT ON table_name TO user_name; - Grants the INSERT privilege on a table to a user.
  • GRANT UPDATE ON table_name TO user_name; - Grants the UPDATE privilege on a table to a user.
  • GRANT DELETE ON table_name TO user_name; - Grants the DELETE privilege on a table to a user.
  • GRANT ALL PRIVILEGES ON table_name TO user_name; - Grants all privileges on a table to a user.

REVOKE

The REVOKE command is used to revoke privileges from users:

  • REVOKE SELECT ON table_name FROM user_name; - Revokes the SELECT privilege on a table from a user.
  • REVOKE INSERT ON table_name FROM user_name; - Revokes the INSERT privilege on a table from a user.
  • REVOKE UPDATE ON table_name FROM user_name; - Revokes the UPDATE privilege on a table from a user.
  • REVOKE DELETE ON table_name FROM user_name; - Revokes the DELETE privilege on a table from a user.
  • REVOKE ALL PRIVILEGES ON table_name FROM user_name; - Revokes all privileges on a table from a user.

COMMIT

The COMMIT command is used to save changes made to the database:

  • COMMIT; - Saves changes made to the database.

ROLLBACK

The ROLLBACK command is used to undo changes made to the database:

  • ROLLBACK; - Undoes changes made to the database.
