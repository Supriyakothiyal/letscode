SQL Joins

INNER JOIN LEFT JOIN RIGHT JOIN FULL JOIN SELF JOIN

INNER JOIN

The INNER JOIN command is used to combine rows from two or more tables based on a matching column:

Example:

SELECT column_name(s)
FROM table1
INNER JOIN table2
ON table1.column_name = table2.column_name;

LEFT JOIN

The LEFT JOIN command returns all rows from the left table and matching rows from the right table. If there is no match, NULL values are used for the right table's columns:

Example:

SELECT column_name(s)
FROM table1
LEFT JOIN table2
ON table1.column_name = table2.column_name;

RIGHT JOIN

The RIGHT JOIN command returns all rows from the right table and matching rows from the left table. If there is no match, NULL values are used for the left table's columns:

Example:

SELECT column_name(s)
FROM table1
RIGHT JOIN table2
ON table1.column_name = table2.column_name;

FULL JOIN

The FULL JOIN command returns all rows from both tables. If there is no match, NULL values are used for the missing columns:

Example:

SELECT column_name(s)
FROM table1
FULL JOIN table2
ON table1.column_name = table2.column_name;

SELF JOIN

The SELF JOIN command is used to join a table with itself:

Example:

SELECT column_name(s)
FROM table1 T1, table1 T2
WHERE T1.column_name = T2.column_name
AND T1.column_name = value;
